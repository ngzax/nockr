# nockr
## Nockr - A Nock interpreter in a RubyGem.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'nockr'
```

And then execute:
```sh
bundle install
```

Or install it yourself into ruby:
```sh
gem install nockr
```

## Testing

### Run all tests, once
```sh
bin/test
```

### Run tests continuously
```sh
bundle exec guard
```
